## Topics
* ### [Requirements](#requirements)

## Requirements
* ### mongoDB
* ### node (version >= 8.11.1)
  ```bash
    $ node - v
    v8.11.1
  ```
* ### npm (version >= 5.6.0)
  ```bash
    $ npm -v
    5.6.0
  ```
## Starting the project
  * ### First install the dependencies
    ```bash
      $ cd ~/my-money-backend && npm install
      $ npm install
    ```
  * ### Finally run the API
    ```bash
      $ npm run production
    ```
